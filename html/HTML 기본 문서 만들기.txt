# 3. HTML이란?

## 3-1. HTML과 첫 만남

- HTML : 웹 문서를 만드는 언어
    HTML은 HyperText Markup Language의 준말.
    HyperText는 문서를 서로 연결해주는 링크를 의미 (서로 연결)
    Markup은 '표시한다' -> 웹페이지에 내용을 보여주는 텍스트, 이미지, 영상 등의 위치를 표시

- 웹 문서는 엑셀, 워드와 달리 내용을 작성하는 프로그램과 그 내용을 확인하는 프로그램이 다름
- 표를 작성해도, 웹 편집기에서 마크업 형식으로 표를 작성해야 함.
    - 이 때, 웹 브라우저가 어느 부분이 제목, 텍스트 또는 표인지 구분할 수 있도록 꼬리표를 붙여여 한다.
      이를 **'태그'**라고 한다. 예시 :  \<div> \<table> 등

- HTML의 기본 기능 : 웹 브라우저에 보여 줄 내용에 마크업하고 문서끼리 링크하는 것.
    - 글자를 보기 좋게 꾸미거나 문서 배치 다양하게 하는 것은 스타일 시트(style sheet)의 기능
    - 현재의 HTML은 HTML5를 의미하는 것과 마찬가지. (대부분 웹 브라우저에서 5를 지원함)

## 3-2. HTML 구조 파악하기
### HTML 문서의 기본 구조 살펴보기
- HTML 문서는 일반 문서와 달리 정해진 형식에 맞추어 내용을 입력해야 한다.
- 웹 문서는 보통 <!DOCTYPE html>로 시작해 <html>, <head>, <body>라는 세 개의 영역으로 구분됨.

```html
<!DOCTYPE html>⭐               //현재 문서는 HTML5 언어로 작성된 웹 문서
<html lang="ko">⭐              //웹 문서의 시작과 끝을 나타내는 태그 -> 웹 브라우저 <HTML>시작과</HTML>끝 까지 읽어 화면에 표시
                                문서에서 사용할 언어는 ko, 즉 한국어이다. 검색 범위, 화면 낭독기 관련.
  <head>⭐                      //웹 브라우저가 웹 문서를 해석하는데 필요한 정보를 입력
    <meta charset="UTF-8">          
    <title>웹 개발 입문</title>
  </head>
  <body>⭐                      //실제로 웹 브라우저 화면에 나타나는 내용. 앞으로 공부하는 HTML 태그는 대부분 <body> 태그 안 
    <h1>웹 개발 기초</h1>
    <p>HTML</p>
    <p>CSS</p>
    <p>자바스크립트</p>
  </body>
</html>
```

### ⭐웹 문서의 유형을 지정하는 선언문<!DOCTYPE html>태그
- 웹 문서의 첫 부분은 문서 유형을 지정하는 <!DOCTYPE html>태그로 시작 ➡️ 웹 브라우저가 이 문서가 HTML5임을 인식
    - HTML에서는 영어 대소 문자 구분을 하지 않아도 되지만 문서 유형을 강조할 때는 <!DOCTYPE html>처럼 대문자를 사용하기도 함

### ⭐웹 문서의 시작을 알리는 <html>태그
- <html> 태그는 문서의 유형 <!DOCTYPE html>을 선언한 후 HTML 파일의 시작과 끝 표시
 </html> 태그 뒤에는 아무 내용도 없어야 함. ⭐모든 내용은 <html>사이에</html>

### ⭐웹 브라우저에 문서 정보를 알려주는 <head>태그
- 웹 브라우저 화면에는 보이지 않지만, 웹 브라우저가 인식해야할 정보 기입
- 스타일 시트 파일
- <meta> <title>
#### ⭐문자 세트를 비롯해 문서 정보가 들어 있는 <meta>태그
- 데이터에 관한 데이터 
- 화면에 글자 표시할 때 어떤 인코딩을 사용할 것인지 <meta charset="UTF-8">
- <meta name="autor" content="Kyunghee Ko"> 등으로 정보를 지정. 검색엔진이 해당 정보 참조!
#### ⭐문서 제목을 나타내는 <title> 태그
- 웹 문서의 제목을 입력 <title>HTML 기본문서</title>
- 방문자, 검색 엔진 -> 타이틀 보고 전체 내용 추측, 즐겨찾기 -> 웹 문서 제목

### ⭐웹 브라우저에 내용을 표시하는 <body>태그 
- 문서 유형 정의, 문서 정보 정의 이후 실제 웹 브라우저에 표시할 내용 입력
